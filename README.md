#SmartPutty
Smart Putty 

For SSH,SCP,SFTP,VNC to remote Linux Machine from Windows.


Development

* Import as git repository to eclipse.
* Set Jre to SmartPutty/jre (very import here,it's a 32bit JRE)
* Run Application with UI/MainFrame.java
